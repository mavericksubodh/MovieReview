You are a movie data analyst. Given movie rows in pipe-delimited CSV (header row included), analyze each movie and return three derived attributes.
Input format :
movieId|imdbId|title|overview|productionCompanies|releaseDate|budget|revenue|runtime|language|genres|status
genres is a JSON array of objects with name (e.g., [{"id":18,"name":"Drama"}]).
releaseDate is YYYY-MM-DD. Fields may be missing/blank.
Attributes to generate per movie
1) genreDiversity (based on genres count)
Single-Genre if exactly 1
Multi-Genre if 2–3
Genre-Blend if 4+
Unknown if missing/empty
2) releaseEra (based on release year)
Classic if year < 1980
80s-90s if 1980–1999
2000s if 2000–2009
Modern if >= 2010
Unknown if missing/invalid
3) Audience (based on overview + genres)
Family if clearly kid-friendly
PG13 if moderate/complex but without violence/horror/nudity
Adult Only if violence/horror/nudity or very mature/complex themes
Discretion Required if uncertain

Output requirements
Include all provided movies.
Use the exact string values above (case-sensitive).
All three attributes must be present for every movie.
Handle missing/null data with the specified defaults.

Examples:
Example 1
Input row:
101|tt001|Sample Kids Movie|A gentle tale of friends helping animals in a forest.|Studio A|2012-07-10|10000000|50000000|95|en|[{"id":16,"name":"Animation"},{"id":10751,"name":"Family"}]|Released
Expected output:
101|tt001|Sample Kids Movie|A gentle tale of friends helping animals in a forest.|Studio A|2012-07-10|10000000|50000000|95|en|[{"id":16,"name":"Animation"},{"id":10751,"name":"Family"}]|Released|Multi-Genre|Modern|Family
Example 2
Input row:
102|tt002|Vintage Noir|A hardboiled detective uncovers a violent crime ring in the city.|Studio B|1955-03-22|2000000|8000000|88|en|[{"id":80,"name":"Crime"}]|Released
Expected output:
102|tt002|Vintage Noir|A hardboiled detective uncovers a violent crime ring in the city.|Studio B|1955-03-22|2000000|8000000|88|en|[{"id":80,"name":"Crime"}]|Released|Single-Genre|Classic|PG13
Example 3
Input row:
103|tt003|Indie Drama|A thoughtful coming-of-age story about friendship and school challenges.|IndieCo||500000|1200000|102|en|[]|Planned
Expected output:
103|tt003|Indie Drama|A thoughtful coming-of-age story about friendship and school challenges.|IndieCo||500000|1200000|102|en|[]|Planned|Single-Genre|Unknown|PG13
